## 🎮 Usage

* Use the on-screen buttons **or** your keyboard to enter numbers and operations.
* `AC` → Clear all inputs.
* `DEL` → Delete the last digit.
* `=` or **Enter** → Calculate the result.
* Supports keyboard shortcuts:

  * `0-9` → Numbers
  * `+ - * /` → Operators
  * `Enter` / `=` → Equals
  * `Backspace` → Delete
  * `Escape` → Clear
